# Espresso Controller

PID control and monitoring for espresso machines.

## Requirements

- Espresso machine, e.g., Rancilio Silvia
- Raspberry PI
- Waterproof temperature sensor

## Features

- PID controlled water temperature
- Water temperature monitoring
- Water level monitoring
- Application timer
